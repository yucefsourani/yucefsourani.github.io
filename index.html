Arboot Fix

Arboot Fix is a simple tool for fix grub bootloader

How To Use

cd && git clone https://github.com/yucefsourani/arboot-fix

su -c "chmod 755 arboot-fix/fedora/arboot-fix.py"

su -c "arboot-fix/fedora/arboot-fix.py"
